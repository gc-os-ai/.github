# GC.OS - German Centre for Open Source AI Software Research

**AI owned by you, governed democratically, for the benefit of society.**

GC.OS (German Centre for Open Source AI Software Research) is dedicated to advancing open source AI software through democratic governance and community-driven development. Our mission is to make AI accessible, transparent, and beneficial for everyone.

For more information, visit [gcos.ai](https://gcos.ai)

## Projects Supported by GC.OS

GC.OS supports the following open source AI projects:

1. **[feature-engine](https://github.com/feature-engine/feature_engine)** - Feature engineering package with sklearn-like functionality for machine learning
2. **[hyperactive](https://github.com/SimonBlanke/Hyperactive)** - An optimization and data collection toolbox for fast prototyping of computationally expensive models
3. **[pgmpy](https://github.com/pgmpy/pgmpy)** - Python library for working with Probabilistic Graphical Models
4. **[prophetverse](https://github.com/felipeangelimvieira/prophetverse)** - A unified interface for time series forecasting models
5. **[ptf](https://github.com/garethmd/pytorchforecasting)** - PyTorch Forecasting for time series forecasting
6. **[pygam](https://github.com/dswah/pyGAM)** - Generalized Additive Models in Python
7. **[pykalman](https://github.com/pykalman/pykalman)** - Kalman Filter implementation in Python
8. **[pyportfolioopt](https://github.com/PyPortfolio/PyPortfolioOpt)** - Financial portfolio optimisation in Python, including classical efficient frontier, Black-Litterman, and Hierarchical Risk Parity
9. **[pytorch-tabular](https://github.com/pytorch-tabular/pytorch_tabular)** - A standard framework for modelling Deep Learning Models for tabular data
10. **[skbase](https://github.com/sktime/skbase)** - Base classes and framework for scikit-learn-compatible packages
11. **[skforecast](https://github.com/JoaquinAmatRodrigo/skforecast)** - Time series forecasting with scikit-learn regressors
12. **[skpro](https://github.com/sktime/skpro)** - Supervised probabilistic prediction in Python
13. **[sktime](https://github.com/sktime/sktime)** - The unified framework for AI with time series
