# GC.OS - German Centre for Open Source AI Software Research

**AI owned by you, governed democratically, for the benefit of society.**

GC.OS (German Centre for Open Source AI Software Research) is dedicated to advancing open source AI software through democratic governance and community-driven development. Our mission is to make AI accessible, transparent, and beneficial for everyone.

For more information, visit [gcos.ai](https://gcos.ai)

## Projects Supported by GC.OS

GC.OS supports the following open source AI projects:

1. **[sktime](https://github.com/sktime/sktime)** - The unified framework for AI with time series
2. **[feature-engine](https://github.com/feature-engine/feature_engine)** - Feature engineering package with sklearn-like functionality for machine learning
3. **[hyperactive](https://github.com/SimonBlanke/Hyperactive)** - An optimization and data collection toolbox for fast prototyping of computationally expensive models
4. **[pgmpy](https://github.com/pgmpy/pgmpy)** - Python library for working with Probabilistic Graphical Models
5. **[prophetverse](https://github.com/felipeangelimvieira/prophetverse)** - A unified interface for time series forecasting models
6. **[ptf](https://github.com/garethmd/pytorchforecasting)** - PyTorch Forecasting for time series forecasting
7. **[pygam](https://github.com/dswah/pyGAM)** - Generalized Additive Models in Python
8. **[pykalman](https://github.com/pykalman/pykalman)** - Kalman Filter implementation in Python
9. **[skbase](https://github.com/sktime/skbase)** - Base classes and framework for scikit-learn-compatible packages
10. **[skforecast](https://github.com/JoaquinAmatRodrigo/skforecast)** - Time series forecasting with scikit-learn regressors
11. **[skpro](https://github.com/sktime/skpro)** - Supervised probabilistic prediction in Python
